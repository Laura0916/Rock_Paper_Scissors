##Rock–Paper–Scissors
A simple, interactive Rock–Paper–Scissors game built with HTML, CSS, and JavaScript.
The project focuses on clean UI, smooth animations, and beginner‑friendly logic.

##Features
-  Play against the computer
-  Animated shaking effect before revealing results
-  Randomized CPU choices
-  Dynamic image switching (rock, paper, scissors)
-  Responsive layout
-  Clean and modern UI using Flexbox

##Project Srtructure
- index.html – Game layout and UI elements
- style.css – Styling, layout, and animations
- script.js – Game logic, CPU randomization, and shake animation

## How It Works
- The player selects rock, paper, or scissors.
- The CPU enters a short shake animation, cycling through all three figures.
- After the animation, the CPU reveals its final choice.
- The game compares both choices and displays the result.

## UI & Animation
The game uses:
- Flexbox for layout
- CSS keyframes for the shaking effect
- JavaScript intervals to cycle images during the animation
This creates a more dynamic and playful experience.
